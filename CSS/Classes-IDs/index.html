Classes & IDs

When should we use classes?

Here are two rules of thumb

  • Ehen you want to apply the same style to multiple elements
  • But not all elements of the same name

A class can be used on muliple elements, and an element can have muliple classes

When should we use IDs?

Here are two rules of thumb

  • When you don't need to apply the same style to muliple elements
  • To have a unique selector to a common element.

A ID cannot be used on muliple elements, and an element cannot have muliple IDs
